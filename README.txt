- Thingiverse: https://www.thingiverse.com/thing:4787450
Gripper Thumb Terminal Device by Skip Meetze and Jon Schull:
- E-NABLE Hub: https://hub.e-nable.org/s/e-nable-devices/wiki/Gripper+Thumb+Terminal+Device
